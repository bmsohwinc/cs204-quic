Asyncio TLS HTTP Demo

This is a static HTML page served over a TCP connection wrapped with TLS using Python's asyncio module.

The goal of this page is simply to provide a body of text with a character count similar to a small example page. The content itself is not important, but we include enough text to make the payload non-trivial and realistic.

Demo information

  • The server listens on localhost using a self-signed certificate.
  • The client connects with TLS enabled and sends a basic HTTP GET request.
  • The server replies with this document and closes the connection.

In a more advanced setting, you might extend this demo to support multiple routes, query parameters, and dynamic responses, but for now we are focusing on a very small and easy to understand example that is still large enough to mimic a realistic scenario.
